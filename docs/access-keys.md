# Enumerate iam access-keys
```
Usage:
  awsrecon access-keys [flags]

Flags:
  -h, --help           help for access-keys
  -k, --key strings    access-keys ids
  -u, --user strings   user names

Global Flags:
  -o, --output string       output filename
      --profile string      AWS profile
      --region strings      AWS regions (default all aws regions)
  -A, --user-agent string   user-agent to use (default "awsrecon")
```
